## About ParkPals

ParkPals is a mobile application that helps users discover, explore, and plan trips to national parks. The app provides features including:

- Interactive park maps
- Wildlife tracking and identification
- Trip planning and packing lists
- Park alerts and news
- Lodging and permit information
- AR camera for wildlife/plant identification
